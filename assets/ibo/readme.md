# IBO Asset Guide

## Book PDFs

Put each book PDF inside `assets/ibo/books/` and use these filenames:

- `trusted-advisor.pdf`
- `crack-the-case.pdf`
- `case-in-point.pdf`
- `the-mckinsey-way.pdf`
- `the-pyramid-principle.pdf`
- `strategy-management.pdf`
- `bcg-on-strategy.pdf`
- `say-it-with-charts.pdf`
- `unfolding-the-napkins.pdf`
- `visualizing-this.pdf`
- `slideology.pdf`

The app will use those paths for the built-in IBO reader and progress tracker.

## Sample Resources

If you want the sample files ready in the repo, place them here:

- `assets/ibo/resources/ibo-syllabus.pdf`
- `assets/ibo/resources/sample-objective-case.pdf`
- `assets/ibo/resources/sample-open-case-analysis.pdf`

## Important Note

The PDF reader works best when the site is served over HTTP or HTTPS, such as Vercel or a local web server. Opening the site directly as a `file://` page may block browser PDF loading.
